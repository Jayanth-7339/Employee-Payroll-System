 Employee Payroll System (Python + MySQL + OOP)
Project Overview

This project is a simple Employee Payroll System developed using Python and MySQL. It manages employee details, salary records, and attendance using a menu-driven program with Object-Oriented Programming.

Features
Add, view, and delete employees
Manage employee salary
Update salary records
Generate payslip
Record employee attendance


Tech Stack
Python 
MySQL 
mysql-connector-python
OOP (Classes & Methods)


Database Structure
employees → employee details
salary → salary records
attendance → attendance data


Workflow
Add Employee
Assign/Update Salary
Track Attendance
Generate Payslip


Key Learnings
Python OOP concepts
Database connectivity (PDBC)
CRUD operations in MySQL
Writing modular code
